Function Reference

_OL_ItemSync

Starts synchronization for all or a single Send/Receive group(s) set up for the user.

#Include <OutlookEX.au3>
_OL_ItemSync($oOL[, $sGroup = ""])

 

Parameters

$oOL Outlook object returned by a preceding call to _OL_Open()
$sGroup Optional: Name of the Send/Receive group to be synchronized (default = all)

 

Return Value

Success: 1
Failure: Returns 0 and sets @error:
    1 - Error returned by method Start. @extended is set to the COM error

 

Remarks

None.
